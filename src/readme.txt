Programmet kan kjøres ved å åpne jar filen som en executable.
Hvis du vil se konsoll mens programmet kjører så kjør kommandoen:
java -jar Equipment.Client

Jeg bruker gson til å lagre meter listen til fil, så for å kjøre applikasjonen i intelliJ må du importere gson-2.8.2 hvis
den allerede ikke er importert.
